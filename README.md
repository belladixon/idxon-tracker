# Reading Tracker App

A full-stack web application that allows users to log reading sessions, track streaks, and view insights over time.

## Features
- Log reading sessions
- View history of sessions
- Track reading streaks
- View summaries and insights

## Tech Stack
- Frontend: React + Vite
- Backend: Express (Node.js)
- Database: Supabase (PostgreSQL)

## Deployment
Prepared for Azure App Service.

## Environment Variables
- SUPABASE_URL
- SUPABASE_ANON_KEY

## How to Run Locally
npm install
npm run build
npm run start
